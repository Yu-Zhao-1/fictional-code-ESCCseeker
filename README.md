# ESCCseeker

## Overview

This repository contains the code for the ESCCseeker project, organized into three main components:

1. **Training ESCCseeker** (`1_training_ESCCseeker.R`)  
   Code for training the ESCCseeker model.

2. **Reproducing performance figures** (`2_performance_figures.R`)  
   Code for reproducing the main performance figures presented in the study.

3. **Running the interception model** (`3_interception_model.R`)  
   Code for running the interception model.
